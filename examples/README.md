# Mackup Configuration Examples

This directory contains example `.mackup.cfg` configurations demonstrating the new features.

## Files

- `mackup-git-backend.cfg` - Git storage backend example
- `mackup-native-sync-filter.cfg` - Native sync filtering example
- `mackup-combined-features.cfg` - Both features combined

## How to Use

Copy the relevant example to your home directory as `.mackup.cfg`:

```bash
cp mackup-git-backend.cfg ~/.mackup.cfg
```

Then customize it for your setup.
